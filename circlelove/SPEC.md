# 《马子潇的恋爱循环》游戏规格说明书

## 1. 项目概述

**项目名称:** 马子潇的恋爱循环
**项目类型:** 视觉小说 / 恋爱模拟游戏
**核心功能:** 多结局恋爱游戏，玩家扮演马子潇与韦淼芊发展关系
**目标用户:** 喜欢日式恋爱游戏的玩家
**预计游戏时间:** 30分钟+

## 2. 视觉 & 渲染规格

### 场景设置
- 主界面：黑色背景 + 漂浮粒子特效
- 游戏界面：中央剧情区 + 右侧路线图 + 底部对话框
- 对话框分为独白框（神态描写）和台词框（对话内容）

### 材质 & 效果
- **UI风格:** 磨砂玻璃（glassmorphism）+ 暗色系
- **漂浮粒子:** 红色/粉色/蓝色渐变球体，缓慢上升消散
- **占有欲高时UI:** 红框呼吸灯闪烁 + 对话框震动效果
- **命案BE滤镜:** 黑白 + 降低亮度 + 褐色色调
- **立绘滤镜:** 占有欲极高时韦淼芊立绘调暗 (brightness 60%)

### 颜色方案
- 主色: #0a0a0f (纯黑)
- 强调色: #ff2e6c (玫红)
- 背景: #12121a (深灰蓝)
- 文字: #e8e8e8 (浅灰白)
- 危险/占有欲: #ff2e63 (亮红)
- JYB金句: #4da6ff (蓝色)
- 好感度: #ff6b9d (粉色)
- 漂浮粒子: 多色渐变

### 交互规格

- **独白时对话框:** 独白显示时对话框隐藏（添加 hidden 类）

## 3. 游戏系统规格

### 数值系统
- **love (好感度):** 初始值 20，最大值 100
- **obsession (占有欲):** 初始值 20，最大值 100
- 动态称呼：当 love >= 50 时：
  - "马子潇" → "宝宝"
  - "韦淼芊" → "宝宝"
  - 主角自称 "小狗"

### 数值影响
| 条件 | 效果 |
|------|------|
| obsession > 80 | 对话框红框震动 (panic-shake动画) |
| obsession > 90 | 韦淼芊立绘 brightness 60% |
| love >= 50 | 启用宝宝称呼模式 |
| love >= 50 | 解锁JYB命案BE分支选项 |

### 结局触发机制
- **选项对错决定结局**（数值仅供参考）
- **好感度>=50后**：JYB出现相关选项出现，选错（帮助JYB）直接触发命案BE
- **特殊BE**: 好感度>=50后出现在特定分支选项，选项添加 `immediate_end` 字段直接结束游戏
- **普通BE/HE**: 通过日常选项累积触发
- **选错立即结束**: 特殊BE选项设置 `immediate_end: "BE_Cage"` 等，跳过加载直接显示结局

### 存档系统
- 6个本地存档位 (localStorage)
- 存档内容: current_node_id, love, obsession, is_baby_mode, timestamp, scene_title, background_name
- 结局存档: GAL_ENDINGS_ARCHIVE (独立键值，不随读档消失)

### JYB金句系统
- **只在JYB在场时触发**
- 位置: 对话框内显示
- 颜色: 蓝色 (#4da6ff)
- 金句列表:
  - "我刻舟早就不为求剑了"
  - "对我流泪吧，我会给你擦的"
  - "白水鉴心"
  - "你就是我的楚门"
  - "爱是场幻觉，我还是戒不掉"

## 5. 交互规格

### 打字机效果
- 速度: 普通对话 20-30字/秒
- 点击屏幕: 立即显示完整文本
- 独白停顿: 显示完后 0.5 秒再允许点击

### 路线图
- 右侧面板显示
- 未解锁节点: 灰色，不可点击
- 已解锁节点: 高亮，可点击跳转

### 分支查看器
- 标题画面新增"分支查看器"按钮
- 思维导图式显示所有分支节点
- 灰色: 未访问节点（不可点击）
- 蓝色: 已访问节点（可点击跳转）
- 粉色高亮: 当前节点
- 显示每个节点选择的选项文本
- 点击节点可跳转到该位置（需确认）

### 音效层级
- BGM: 背景音乐 (循环)
- SE: 环境音效 (一次性: 开门声、心跳声等)
