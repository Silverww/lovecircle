## 快乐结局（Happy Ending - HE）

### HE_1 - 甜蜜日常 
**类型**: 普通快乐结局
**达成条件**: 稳定关系、互相珍视
**选择路线**:
```
开始 → 第一次见面（任意选择）→ 表白（"我愿意"）
→ 天台约定（回应感情或牵紧手）→ HE_route（选择"紧紧握住她的手"）
→ HE_1【结局】
```
**关键节点**: HE_route  
**关键选项**: 选择"（紧紧握住她的手）"
**推荐统计**: love >= 60, obsession <= 40

---

### HE_2 - 深夜约定  
**类型**: 甜蜜承诺结局
**达成条件**: 深化关系、天台的特殊约定
**选择路线**:
```
开始 → 确认恋爱关系 → 留下陪她（选择"好"）
→ 度过相处时光 → 晚间谈心（选择"你这样让我很放心"或类似）
→ rooftop_night_promise（选择"我们一辈子在一起"）
→ HE_2【结局】
```
**关键节点**: rooftop_night_promise（需创建）
**关键选项**: 选择浪漫承诺的对话
**推荐统计**: love >= 70, obsession <= 30

### HE_3 - 毕业旅行  
**类型**: 冒险与回忆
**达成条件**: 关系稳定、一起规划未来
**选择路线**:
```
开始 → 确认恋爱 → 接受旅行邀约（选择"好啊，去哪里"）
→ 海边旅行中 → 海滩夕阳（选择"我也是"）
→ 在beautiful moment下选择"我的也是"

→ HE_3【结局】
```
**关键节点**: park_ending  
**关键选项**: 选择"我也是"  
**推荐统计**: love >= 65, obsession <= 35

---

→ 但仍选择"我只是需要时间适应" → normal_path后续

```
**关键节点**: cohabitation_decision（需创建）  
**关键选项**: 选择"我们一起租房吧"
**推荐统计**: love >= 55, obsession <= 50, 必须拒绝cage_path

---

### HE_5 - 永久契约  
**类型**: 婚礼、终身承诺
**达成条件**: 极高的love值、稳定的关系、接受戒指
**选择路线**:
```
开始 → 确认恋爱 → 接受旅行 → 海滩确认

## 坏结局（Bad Ending - BE）

### BE_Cage - 笼中鸟  
**类型**: 特殊BE - 被囚禁的爱
**达成条件**: 接受被"关起来"、执念过高
**选择路线**:
```
开始 → 确认恋爱 → 选择"好"留下
→ cage_dialogue_1中选择"为什么"追问（obsession+10）
→ cage_approach_1中选择"我愿意试试"
→ cage_path（进入房间）
→ caged_life（选择"微微点头"）
→ BE_Cage【结局】
```
**关键节点**: caged_life  
**关键选项**: 选择"（微微点头）"
**触发条件**: obsession >= 70, 必须每步都选择顺从
**推荐统计**: obsession >= 80, love可以较低

---

### BE_Loop - 自杀循环  
**类型**: 特殊BE - 无尽痛苦循环
**达成条件**: 无法承受她的情感压力、选择冷漠拒绝
**选择路线**:
```
开始 → 确认恋爱 → 旅行邀约中选"我想先和CZH聚聚"
→ 触发嫉妒反应 → 后续冲突升级
→ loop_breakdown（她表达"没有你我活不下去"）
→ 选择"你这样让我压力很大"
→ BE_Loop【结局】
```
**关键节点**: loop_breakdown  
**关键选项**: 选择"你这样让我压力很大"（不要选择拯救她）
**触发条件**: 在critical moment选择逃避
**推荐统计**: love <= 40, obsession可以较高

---

### BE_Murder - 命案  
**类型**: 命案BE - 无法挽回的悲剧
**达成条件**: 高执念、与朋友走近、激怒她
**选择路线**:
```
开始 → 确认恋爱 → 旅行邀约选"我想先和CZH聚聚"
→ 持续与CZH互动 → 在嫉妒反应中选择多次拒绝
→ 她发现你与朋友接触 → murder_reveal（警察来敲门前）
→ 选择"你做了什么" → morning_after_murder
→ 选择"（震惊）" → BE_Murder【结局】
```
**关键节点**: murder_reveal, morning_after_murder  
**关键选项**: 任何都是"（恐惧）"或"（震惊）"
**触发条件**: obsession >= 75, 多次无视她的嫉妒
**推荐统计**: obsession >= 80, 与朋友互动多次

## 游戏统计系统

### 主要属性
- **Love（爱值）**: 0-100，代表对她的爱与好感
  - 0-30: 冷漠
  - 31-50: 普通好感  
  - 51-70: 深厚好感
  - 71-100: 深爱

- **Obsession（执念值）**: 0-100+，代表她对你的执着和占有欲
  - 0-30: 正常爱
  - 31-60: 开始执着
  - 61-80: 严重执念
  - 81+: 极端执念

### 特殊标记
- **isBabyMode**: 当 love >= 50 时触发，改变对话文本
- **UnlockedNodes**: 追踪已解锁的场景
- **VisitedNodes**: 追踪已访问过的场景

## 高级建议

### 如何快速达成特定结局
1. **保存档案**: 在关键选择点保存（使用localStorage）
2. **重复游玩**: 尝试不同的选择路线以触发不同分支
3. **属性监控**: 在控制台输入 `console.log(gameState)` 查看当前属性
4. **隐藏选项**: 某些特殊选项只在特定条件下出现（如love或obsession处于特定范围）

### 隐藏内容
- **Baby Mode**: love >= 50时触发，改变韦淼芊的对话方式（更亲密）
- **特殊分支**: 某些节点只在love和obsession特定比例时出现
- **CG解锁**: 部分结局包含特殊的场景描写
